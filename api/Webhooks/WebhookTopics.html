Webhook Topics

Webhook topics define the types of events you can subscribe to. Each topic represents a specific event type in the Xemelgo system.

Topic Format​

Topics follow a hierarchical naming convention: {resource}.{action} (e.g., inventory.cycle_count).

Available Topics​

TopicDescription
inventory.cycle_countTriggered when an inventory cycle count is submitted

Additional topics will be documented as they become available. Contact your Xemelgo representative for information about upcoming topics.

Generic Webhook Events​

All webhook events follow a standard structure with the following fields:

{
  "eventId": "c1314cc5-60f7-ddba-7be3-900030a8ee05",
  "topic": "inventory.cycle_count",
  "payload": {}
}

Event Structure​

PropertyTypeDescription
eventIdStringWebhook event UUID (unique identifier)
topicStringWebhook topic that identifies the event type
payloadObjectTopic-specific payload containing event data

Topic Specific Payloads​

inventory.cycle_count​

Occurs whenever an inventory cycle count is submitted. Provides item level information and aggregate counts.

Payload Structure​

{
  "eventId": "c1314cc5-60f7-ddba-7be3-900030a8ee05",
  "topic": "inventory.cycle_count",
  "payload": {
    "uuid": "6ccd5f10-31e9-6a7e-1005-638be25f786a",
    "creationDate": 1764126384655,
    "locationId": "WEBHOOK_ZONE1",
    "customerId": null,
    "submittedByUser": {
      "alias": "WEBHOOK_APIS_PERSON_1",
      "firstName": "WEBHOOK_APIS_GIVEN_1",
      "lastName": "WEBHOOK_APIS_FAMILY_1"
    },
    "inventoryParts": [
      {
        "id": "TypeA",
        "name": "Type A Name",
        "detectedCount": 1,
        "movedCount": 1,
        "addedCount": 1,
        "foundCount": 1,
        "missingCount": 1,
        "removedCount": 1,
        "noActionCount": 1,
        "totalCount": 7,
        "inventory": [
          { "action": "DETECTED", "trackerSerial": "012345", "id": "item-a1" },
          { "action": "MOVED", "trackerSerial": "012346", "id": "item-a2" },
          { "action": "ADDED", "trackerSerial": "012347", "id": "item-a3" },
          { "action": "FOUND", "trackerSerial": "012348", "id": "item-a4" },
          { "action": "MISSING", "trackerSerial": "012349", "id": "item-a5" },
          { "action": "REMOVED", "trackerSerial": "012350", "id": "item-a6" },
          { "action": "NONE", "trackerSerial": "012351", "id": "item-a7" }
        ]
      }
    ]
  }
}

Payload Properties​

PropertyTypeDescription
uuidStringUnique identifier for the cycle count event
creationDateNumberTimestamp when the cycle count was created (milliseconds)
locationIdStringLocation where the cycle count was performed
customerIdStringCustomer identifier (null if not applicable)
submittedByUserObjectInformation about the user who submitted the cycle count
inventoryPartsArrayArray of inventory parts with their counts and item details

submittedByUser Properties​

PropertyTypeDescription
aliasStringUser alias/username
firstNameStringUser's first name
lastNameStringUser's last name

inventoryParts Properties​

PropertyTypeDescription
idStringInventory part/item type identifier
nameStringDisplay name of the inventory part
detectedCountNumberCount of items detected during cycle count
movedCountNumberCount of items that were moved
addedCountNumberCount of items that were added
foundCountNumberCount of items that were found
missingCountNumberCount of items that were missing
removedCountNumberCount of items that were removed
noActionCountNumberCount of items with no action taken
totalCountNumberTotal count of items for this part
inventoryArrayArray of individual inventory items with their actions

inventory Item Properties​

PropertyTypeDescription
actionStringAction taken for this item (Detected, MOVED, ADDED, FOUND, MISSING, REMOVED, NONE)
trackerSerialStringRFID tracker serial number for the item
idStringUnique identifier for the inventory item
